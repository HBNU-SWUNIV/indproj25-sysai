## Project Background
- ### 필요성
  - **사용자 맞춤형 대시보드의 필요성**  
    - 기존 모니터링 시스템은 고정된 인터페이스로 인해 사용자가 원하는 데이터를 자유롭게 구성하기 어렵고, 실시간 분석 및 시각화 기능이 부족하다.  
    - 본 프로젝트는 **사용자 맞춤형 컴포넌트 배치와 유연한 UI 구성**을 지원하여 관리자가 필요한 정보를 직관적으로 확인할 수 있도록 한다.
  
  - **실시간 데이터 분석 및 시각화 강화**  
      - 기존 시스템의 정적 데이터 표시 한계를 개선하여, **실시간 분석 결과를 다양한 형태(차트·그래프·스켈레톤 등)** 로 제공한다. 이를 통해 환자의 상태를 즉시 파악하고 효율적으로 관리할 수 있다.
  
  - **직관적 UI/UX 및 사용자 편의성 향상**  
      - 의료진·보호자·관리자 등 다양한 사용자가 쉽게 접근할 수 있도록 **데이터 자유 배치형 대시보드**를 제공한다.  
      - 불필요한 정보를 최소화하고, 그래프·아이콘·색상 변화를 통해 정보 가독성을 높인다.

## Project Outcome
- ### 학술대회 발표
  ## [2025 전기전자학회]
  - 실시간 데이터 시각화를 위한 모듈식 파이프라인 아키텍처 및 자동 코드생성(한소연) 
  - 플러그인 대시보드 지연로딩 설계 및 렌더링 적재 방안 연구(서진주)
  - 플러그인 기반 대시보드 시스템에서의 실행 전략 유형에 따른 시뮬래이션 모델링 및 성능 분석(김영빈)

  ## [2025 춘계공동학술대회]
  - 시뮬레이션 기반 실시간 데이터 시각화를 위한 아키텍처 및 방법론 제안(한소연)
  - 이산사건 시스템 형식론 기반 대시보드 프론트엔드 설계 및 구현 (서진주)
  - 이산사건 시스템 형식론을 활용한 유연한 대시보드 서버 설계 및 구현(김영빈)
